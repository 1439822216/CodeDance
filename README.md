一、目录结构
-CodeDance
 |
 |--bak--原项目的备份文件
 |
 |--video
 |  |
 |  |--test.mp4--项目测试使用的原视频文件
 |  |
 |  |--test-code.mp4--原视频文件生成的代码式视频文件
 |
 |--video_2_code_video.py--实现代码

二、环境
操作系统： Windows
Python版本： Python3.7
依赖库： pip install opencv-python pillow
依赖程序： ffmpeg（路径需加入PATH环境变量）

三、使用
在config_dict中配置好输入的视频文件路径，直接运行video_2_code_video.py即可
